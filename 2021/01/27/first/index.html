函数可变参数

在参数名前面的*表示args是一个可变参数
def add(*args):
total = 0
for val in args:
total += val
return total

模块定义

可执行代码放在*if name == ‘main‘**:条件语句下，导入该模块时，由于模块名称不是main***，因此不会执行该条件成立时的代码
只有直接运行该模块时，该条件下代码才会被执行
一般书写格式：
def main():
# Todo: Add your code here
pass

if __name__ == '__main__':
    main()
